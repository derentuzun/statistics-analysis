# England Demographic Analysis — 2021 Census

Exploratory analysis of age distribution patterns across England 
using UK 2021 Census data.

## Overview
Analysed demographic data across 33,755 Lower Layer Super Output 
Areas (LSOAs) in England to investigate population age patterns, 
regional differences, and unusual demographic profiles.

## What I did
- Calculated age proportions and estimated median age for each LSOA
- Compared age distributions across area types: London Boroughs, 
  Metropolitan Districts, Unitary Authorities, and Non-metropolitan Districts
- Classified LSOAs into demographic profiles (e.g. student areas, 
  retirement areas, working-age areas) using scatter plot analysis
- Produced geospatial visualisations using GeoPandas

## Key Findings
- London Boroughs show significantly younger age profiles 
  than rural Non-metropolitan Districts
- Student and retirement communities form distinct demographic clusters
  visible in younger vs older proportion scatter plots
